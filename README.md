# Zomato Market Intelligence & Expansion Strategy System

##  Objective
To analyze Zomato restaurant data and identify optimal locations, pricing strategies, and cuisine trends for business expansion.

## Tools Used
- Python (Pandas, NumPy)
- Data Analysis
- Feature Engineering

##  Key Analysis
- Price vs Rating Analysis
- Location-based Demand & Quality Analysis
- Online Order Impact
- Cuisine Trend Analysis

##  Key Insights
- High-priced restaurants tend to have better ratings
- BTM shows high demand but lower ratings (saturated market)
- Koramangala 5th Block offers best balance of demand and quality
- Premium multi-cuisine restaurants have highest ratings

## Expansion Strategy
A scoring model was developed combining demand and ratings to identify high-potential locations.

## Project Structure
- zomato_analysis.ipynb
- dataset.csv

## Conclusion
This project provides data-driven insights for strategic decision-making in restaurant expansion.
